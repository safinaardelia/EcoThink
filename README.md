# EcoThink
EcoThink adalah media pembelajaran IPA berbasis komputasi pada materi Pencemaran Lingkungan untuk siswa SMP. Media ini mengintegrasikan materi, simulasi, visualisasi data, dan evaluasi interaktif guna mendukung pengembangan literasi sains, computational thinking, serta pemahaman peserta didik terhadap isu lingkungan yang relevan dengan SDGs.
